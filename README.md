## 总体要求

- **实验**：4 个大类（9.1–9.4）共 10 个小实验，至少完成 3 个且分属不同大类；良 4 个；优 5 个以上。
- **课设**：完成 9.5（银河麒麟适配迁移）或 9.6（虚拟化技术与应用），需按完整项目要求实现。

## 技术栈

- 银河麒麟高级服务器操作系统（工业版）V10
- QEMU / KVM 虚拟化
- Docker 容器化
- C / 内核模块 / FUSE 文件系统
- DEB / RPM 打包规范
